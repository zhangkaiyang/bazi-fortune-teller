# 部署指南

## 🚀 快速部署

### 方式一：使用 Docker Compose（推荐）

```bash
# 克隆项目
git clone https://github.com/zhangkaiyang/bazi-fortune-teller.git
cd bazi-fortune-teller

# 复制环境变量文件
cp .env.example .env
# 编辑 .env 文件，设置 SECRET_KEY 等变量

# 启动服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f
```

服务访问地址：
- 前端：http://localhost:3000
- 后端 API：http://localhost:8000
- API 文档：http://localhost:8000/docs
- 数据库：localhost:5432 (用户名: postgres, 密码: password)
- Redis：localhost:6379

### 方式二：手动部署

#### 1. 后端部署
```bash
cd backend

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
export DATABASE_URL=postgresql://user:password@localhost:5432/bazi
export SECRET_KEY=your-secret-key-here

# 启动服务
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

#### 2. 前端部署
```bash
cd frontend

# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 预览生产版本
npm run preview
```

#### 3. 数据库初始化
```sql
-- 创建数据库
CREATE DATABASE bazi;

-- 创建用户表（示例）
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    hashed_password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 创建查询记录表
CREATE TABLE bazi_queries (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    birth_date DATE NOT NULL,
    birth_time TIME NOT NULL,
    gender VARCHAR(10) NOT NULL,
    bazi_result JSONB NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 📊 数据库迁移

使用 Alembic 进行数据库迁移：

```bash
cd backend

# 初始化 Alembic
alembic init migrations

# 创建迁移
alembic revision --autogenerate -m "Initial migration"

# 应用迁移
alembic upgrade head

# 回滚迁移
alembic downgrade -1
```

## 🔄 CI/CD 配置

### GitHub Actions 示例
```yaml
# .github/workflows/deploy.yml
name: Deploy

on:
  push:
    branches: [ main ]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run tests
        run: |
          cd backend
          pip install -r requirements.txt
          pytest

  deploy:
    needs: test
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Deploy to Railway
        uses: railwayapp/action@v1
        with:
          railway_token: ${{ secrets.RAILWAY_TOKEN }}
          service: bazi-backend
```

## 🚨 故障排除

### 常见问题

1. **数据库连接失败**
   ```bash
   # 检查数据库服务
   docker-compose ps
   
   # 检查日志
   docker-compose logs db
   
   # 测试连接
   psql -h localhost -U postgres -d bazi
   ```

2. **前端无法访问 API**
   ```javascript
   // 检查环境变量
   console.log(import.meta.env.VITE_API_URL)
   
   // 检查 CORS 配置
   // backend/main.py 中的 CORS 设置
   ```

3. **八字计算错误**
   ```bash
   # 检查日志
   docker-compose logs backend
   
   # 测试 API
   curl http://localhost:8000/health
   curl -X POST http://localhost:8000/api/bazi/calculate \
     -H "Content-Type: application/json" \
     -d '{"birth_date":"1990-01-15","birth_time":"08:30","gender":"male"}'
   ```

### 性能优化

1. **启用缓存**
   ```python
   # 使用 Redis 缓存计算结果
   from redis import Redis
   redis = Redis.from_url(REDIS_URL)
   ```

2. **数据库索引**
   ```sql
   CREATE INDEX idx_bazi_queries_user_id ON bazi_queries(user_id);
   CREATE INDEX idx_bazi_queries_created_at ON bazi_queries(created_at);
   ```

3. **分页查询**
   ```python
   # 大型结果集使用分页
   skip = (page - 1) * limit
   limit = page_size
   ```
