# 八字算命网站 - 项目详细设计

## 🎯 项目概述
**项目名称**：八字命理分析系统  
**核心功能**：输入出生年月日时，输出八字排盘、简单解析和年度运势

## 📋 功能需求

### 1. 核心功能
- **八字排盘**：根据出生时间计算八字（年柱、月柱、日柱、时柱）
- **简单解析**：对八字五行、十神、格局进行基础分析
- **年度运势**：分析当前年份的运势（流年、大运）
- **历史记录**：保存用户的查询记录（可选）

### 2. 扩展功能
- **五行平衡分析**
- **十神关系图解**
- **喜用神建议**
- **配对分析**（两人八字合婚）
- **每日运势**（基于八字）

## 🔢 核心算法设计

### 1. 八字计算逻辑
```python
# 伪代码
def calculate_bazi(birth_datetime, gender, use_solar_time=True):
    # 1. 转换为农历
    lunar_date = solar_to_lunar(birth_datetime)
    
    # 2. 计算年柱
    year_stem, year_branch = get_year_pillar(lunar_date.year)
    
    # 3. 计算月柱（考虑节气）
    month_stem, month_branch = get_month_pillar(lunar_date, solar_term)
    
    # 4. 计算日柱（公式计算）
    day_stem, day_branch = get_day_pillar(birth_datetime)
    
    # 5. 计算时柱
    hour_stem, hour_branch = get_hour_pillar(day_stem, birth_datetime.hour)
    
    return {
        'year': f"{year_stem}{year_branch}",
        'month': f"{month_stem}{month_branch}",
        'day': f"{day_stem}{day_branch}",
        'hour': f"{hour_stem}{hour_branch}"
    }
```

### 2. 五行分析
```python
def analyze_five_elements(bazi):
    elements = {
        '金': 0, '木': 0, '水': 0, '火': 0, '土': 0
    }
    
    # 统计八字中的五行
    for pillar in bazi.values():
        stem_element = get_element_from_stem(pillar[0])
        branch_element = get_element_from_branch(pillar[1])
        elements[stem_element] += 1
        elements[branch_element] += 1
    
    return elements
```

### 3. 十神分析

def analyze_ten_gods(day_master, bazi):
